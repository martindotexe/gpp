# GPP - Go Pretty Print (because PP was taken by pearl...)

Go Pretty print displays images in the terminal. It supports these formats:

- [x] JPG
- [x] PNG
- [x] GIF (first frame)
- [ ] GIF (animated)
- [ ] WebP
- [ ] BMP
- [ ] [Netpbm](https://en.wikipedia.org/wiki/Netpbm) (PPM, PGM, PBM, and PNM)
